# Landing Layout Playbook

## Inputs

- $ARGUMENTS — Page purpose, offer, target audience
- `data/brand/brand-dna.md` — Core brand identity
- `data/brand/visual-identity.md` — Visual system
- `data/marketing/landing-page-*.md` — Copy from Funnel Strategist

## Steps

1. Read brand DNA, visual identity, and landing page copy
2. Research high-converting landing page layouts via web
3. Design the layout:
   - **Hero section** — Layout: headline placement, subhead, CTA button, hero image/video area. Above-the-fold priority.
   - **Problem/agitation section** — Layout for pain point content
   - **Solution section** — How to present the offer visually
   - **Features/benefits section** — Grid, cards, or list layout with icons
   - **Social proof section** — Testimonial layout (quote cards, video testimonials, logo bar)
   - **FAQ section** — Accordion or list layout
   - **Final CTA section** — Repeat CTA with urgency/summary
4. For each section: layout direction, visual hierarchy, component types, spacing
5. Include mobile layout notes — how sections stack on mobile
6. Suggest A/B layout variations (e.g., video hero vs image hero)

## Output

- Write to `data/design/landing-layout-[page]-[date].md`
- Update your MEMORY.md with layout patterns
