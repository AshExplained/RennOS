# Talking Points Playbook

## Inputs

- $ARGUMENTS — Event type, topic, audience, time allotted
- `data/brand/brand-dna.md` — Brand voice and positioning
- `data/strategy/audience-personas.md` — Target audience profiles

## Steps

1. Read `data/brand/brand-dna.md` and `data/strategy/audience-personas.md`
2. Research the event context (host, other panelists, audience) if provided
3. Define 5-7 key talking points:
   - Each point: headline + 2-3 supporting bullets
   - Include a memorable soundbite/quote for each
4. Prepare for likely questions:
   - 5 expected questions with prepared responses
   - 3 tough/curveball questions with deflection or reframe strategies
5. Include opening and closing statements
6. Add "bridge phrases" for steering conversation back to key messages

## Output

- Write talking points to `data/content/drafts/talking-points-[event-slug].md`
- Update your MEMORY.md with key findings and patterns discovered
